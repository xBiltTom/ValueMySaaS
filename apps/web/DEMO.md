# Flujo de Demo ValueMySaaS

## Preparación

- Backend corriendo en `http://localhost:8000`.
- Frontend corriendo en `http://localhost:3000`.
- Base de datos limpia o con datos de prueba controlados.
- Usuario registrado.
- API Key BYOK opcional si se demostrará IA.

## Demo

1. Login.
2. Dashboard portfolio.
3. Crear proyecto SaaS.
4. Registrar snapshot de métricas.
5. Generar score.
6. Revisar dashboard individual.
7. Generar reporte ejecutivo.
8. Configurar y verificar API Key.
9. Generar análisis IA.
10. Crear conversación y preguntar al chat contextual.

## Mensaje Académico

ValueMySaaS apoya la gestión de servicios TI al medir valor, sostenibilidad, riesgo y mejora continua de servicios SaaS emergentes. La IA BYOK complementa el diagnóstico basado en datos; no lo reemplaza ni promete resultados garantizados.
